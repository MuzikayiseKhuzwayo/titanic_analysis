# Titanic Analysis

This project holds the analysis and modeling for the Titanic dataset. Heavy model training and large-scale executions are intended to run elsewhere (e.g., in a dedicated compute environment). This repository serves as the centralized version control for notebooks, source code, and configurations.

## Directory Structure

- `data/`: Contains raw and processed data. (Note: Actual data files are git-ignored).
- `notebooks/`: Jupyter notebooks for exploratory data analysis and modeling.
- `src/`: Reusable Python source code (scripts, modules) imported by notebooks or used in production.

## Getting Started

1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Launch Jupyter: `jupyter notebook` or `jupyter lab`.
